¿Programar? ¿Que significa?

El desarrollo de software junto al armado de un sitio o aplicacion web es uno de los empleos mas demandados del mundo. Donde multiples empresas buscan desarrolladores capacitados para cumplir el avance tecnologico. Debido a la alta demanda, es tambien, los trabajos mejor pagos. Ahora bien, que es programar.

En terminos sencillos es darle las instrucciones que la maquina pueda entender para resolver un problema especifico. Mediante los correctos pasos logicos se puede llegar a diferentes soluciones. Con el gran avance tecnologica, hoy en dia, las computadoras son capaces de solucionar problemas de gran dificultad. Las instrucciones deben estar en el lenguage maquina, la cual son los lenguajes de programacion.

Lenguajes de programacion

Existen dos clases de programacion. Aquellos de alto nivel y otros de bajo nivel. Donde el primer son los que se acerca mas al lenguaje humano, es decir las instrucciones se asemejan mas al idioma que nosotros utilizamos al comunicarnos entre nosotros. Entre estos lenguajes estan todos los que vamos a visualizar en esta pagina web. El segundo, sin embargo, es aquel mas cercano al lenguaje maquina, donde este maneja las millones de combinaciones de 0 (ceros) y 1 (unos). Debido a su alta complejidad en entender lo omitiré. Assembly language es un buen ejemplo de un lenguaje de bajo nivel.

Python es uno de los tres lenguajes mas usados en el mundo, subiendo e intercalandose posiciones por el primer lugar. Python se caracteriza por su rapidez, su simplicidad y la curva de aprendizaje tan amigable y cercano al desarrollador. Este lenguaje creado por el informatico Guido Van Rossum en 1989 es interpretado, paradigmatico y de multiplataforma.

  • Interpretado: Ejecuta el programa directamente, instruccion a instruccion. El interprete se encarga de leer una a una las instrucciones textuales del programa conforme estas necesitan ser ejecutadas y descomponerlas en instrucciones del sistema, además se encarga de automatizar algunas de las tareas típicas de un programador como declaraciones de variables o dependencias, de esta manera el proceso de programar se suele agilizar mucho lo cual repercute en la eficiencia del que tiene que escribir el código.
  • Multiparadigma: Es un lenguaje que admite varios paradigmas de programacion (modelos de desarrollo), variando el estilo de programacion. Estos son Programacion orientada a objetos (POO), programacion imperativa y programacion funcional.
  • Multiplataforma: Es capaz de ejecutarse en diferentes sistemas operativos como Linux, Unix, MacOS y windows.

Una encuesta realizada en 2019 por el propio Python arrojó que el 59% de los desarrolladores usa Python para Data Analysis.

En la actualidad existen muchas empresas que usan Python, unos de los cuales estan Google, Instagram, Netflix entre otros.

Habilidades mas buscadas de un programador de Python

  • Frameworks: Dominar el desarrollo con frameworks( estructuras de trabajo ) como Django, Flask, Pyramid, Pylons.
  • Base de datos: Dominar la base de datos relacionales. La cual son aquellas que almacenan y proporcionan accesos a puntos de datos relacionados entre si. Esto aumenta la capacidad de construir sistemas de datos que prevalecen en el tiempo. Base de datos como PostgreSQL, MySQL, SQLite, mongoDB y otros
  • Programacion orientada a objetos: Dominar este paradigma de programacion es uno de los primeros requisitos en este lenguaje. La cual consiste en organizar el codigo en clases y crear objetos que se relacionan entre si para alcanzar la solucion del programa.
  • Continuo aprendizaje autodidacta: A igual que el resto de lenguajes de programacion, la cual requieren que el programador investigue las nuevas mejoras, actualizaciones o cambios que la comunidad de desarrolladores hagan sobre dicho lenguaje u otros factores. Ya que es comun que el gran tamaño de la comunidad realicen sus aportaciones al codigo abierto de python y asi su evolución.
  • Trabajo en equipo: La mayor parte de los proyectos en el desarrollo de software se hacen con colaboraciones con otros programadores. Y esto sucede en la mayoria de las empresas. Por eso es importante conocer las metodologias de trabajo y conocer las formas adecuadas y correctas de trabajar en un equipo.

Java es un lenguaje orientado a objetos. Con una curva de aprendizaje baja. Este lenguaje es uno de compiten por liderar el escalafon de los lenguajes de programacion mas populares del mundo. Ofrece un codigo robusto, la cual permite un manejo automatico de la memoria reduciendo la cantidad de errores. Es interpretado, multiplataforma al igual que Python y PHP entre otros. Ademas posee otras caracteristicas importantes:

  • Simplicidad: Su diseño base parte de c++ pero con los cambios que lo hace mas facil y rapido de entender y aprender. Elimina la aritmetica de punteros (referencia a un lugar especifico de la memoria para realizar alguna operacion con otra direccion de memoria), referencias, registros (struct), definicion de tipos (typedef), macros (#define), el control de liberar memoria(la cual implementa el reciclador de memoria dinamica).
  • Orientado a objetos: Java implementa la tecnologia de c++ con muchas modificaciones pero manteniendo la simplicidad del lenguaje. Soporta las caracteristicas de un lenguaje orientado a objetos: encapsulacion, herencia y polimorfismo. Java incorpora la resolucion dinamica de métodos que define la interaccion entre objetos excluyendo variables de instancias o implementacion de metodos.
  • Seguro: El codigo Java pasa muchos tests antes de ejecutarse en una maquina. Estos test es un verificador de byte-codes que comprueba el formato de fragmentos de codigo y detecta codigo ilegal (codigo punteros falsos, violando los derechos de acceso sobre objetos) Ademas posee un cargador de clases, la cual separa las clases locales y externas. Donde estas ultimas no puede suplantar una clase local predefinida
  • Multihilo: Java permite muchas actividades simultaneas en un programa. Para entenderlo mejor, los hilos son pequeños procesos. Entonces al realizar multiples procesos mejora el rendimiento interactivo y mejora el comportamiento en tiempo real.

Java es muy utilizado para el desarrollo de juegos. Dentro de las cuales el mas popular es Minecraft, donde junto con jMOnkeyEngine se pueden desarrollar juegos muy interesantes.

Habilidades mas buscadas de un programador de Java

  • Marcos de java: Cuerpos de codigo escrito previamente, la cual se utilizan para crear aplicaciones con Java, entre los que estan Quarkus, Grails, Hibernate, Google Web Toolkit (GWT), entre otros.
  • Herramientos de construccion de java: Son herramientas para realizar tareas mecanicas y repetitivas, normalmente durante la fase de compilacion y construccion(build),es decir es un software para procesos de automatizacion de compilacion. Por ejemplo Ant y make cumplen esa funcionalidad.
  • Contenedores y servidores de aplicaciones java: EL primero son albergadores de Servlets(aplicaciones de Java) y el segundo permite ejecutar aplicaciones con EJB(API de java que implementa diferentes servicios).
  • Herramientas de prueba java: Son frameworks de tests para probar aplicaciones Java

Javascript es, sin duda, uno de los lenguajes mas usados. Conocido como aquella que le otorga funciones dinamica (comportamientos) a la web mediante scripts. Es un lenguaje, donde si bien es usado del lado del cliente en los sitios web, tambien se ejecuta del lado del servidor junto al framework NodeJs. Es un lenguaje simple, multiparadigma, interpretado, dinamico, de tipado debil y basado en el estandar EcmaScript entre otras caracteristicas.

C# es un lenguaje de programacion multiparadigma desarrollado y estandarizado por Microsoft como parte de la plataforma su .NET. C# es uno de los lenguajes de programación diseñados para la infraestructura de lenguaje común (common Language Infractructure o CLI). Su sintaxis deriva de C/C++ y utiliza el modelo de objetos de la plataforma .NET. C# es multiplataforma, multiparadigma ( estructurado, orientado a objetos, imperativo, funcional, reflexivo y dirigido por eventos). Al igual que C y c++, c# se compila, es decir, que se genera un código intermedio llamado MSIL (Microsoft Intermediate Language) y luego ese código se vuelve a compilar para obtener el formato nativo en la máquina en la que se ejecuta. Entre sus principales caracteristicas estan:

  • Tipado fuerte: En c# debes declarar los tipos de datos con las variables, siendo estas no globales.
  • Orientado a componentes: Podemos definir propiedades sin necesidad de crear metodos, como en java, o usar eventos sin tratar con punteros a funciones.
  • Multihilo: Puedes dividir tu código en múltiples hilos de ejecución, trabajar en paralelo y sincronizándose al final.
  • Mejora en la gestión de memoria: Al igual que Java, en C# dispone de un recolector de basura que destruye los objetos que no se usan en memoria.
  • Seguridad de tipos: C# incluye mecanismos de control de acceso a tipos de datos, lo que garantiza que no se produzcan errores difíciles de detectar como un acceso a memoria de ningún objeto, por ejemplo. Para ello, el lenguaje provee de una serie de normas de sintaxis, como por ejemplo no realizar conversiones entre tipos que no sean compatibles. Además, no se pueden usar variables no inicializadas previamente, y en el acceso a tablas se hace una comprobación de rangos para que no se excedan ninguno de los índices de la misma.

El lenguaje de programación C# facilita y permite la creación de una gran variedad de aplicaciones empresariales, así como también para la creación de juegos para Windows, Xbox360 y Android. C# es el lenguaje de programación utilizado en Unity. Aprende a escribir scripts simples de C# para crear elementos de juego, mecánicas, elementos visuales, sistemas de inteligencia artificial y más.

Habilidades mas buscadas de un programador de C#:

  • .NET: Conocer de C#, significa que estas aprendiendo del marco de desarrollo de software de Microsoft, .NET. Al igual que lenguajes como c++ y visual basic. Si bien ahondaremos mas en la seccion de visual studio.
  • Programacion asincronica: C# tiene un modelo de programación asincrónico de nivel de lenguaje que permite escribir fácilmente código asincrónico sin tener que hacer malabares con las devoluciones de llamada o ajustarse a una biblioteca que admita la asincronía.
  • Manejo de eventos: C# permite la manipulacion de eventos como javascript. Sin embargo se basan en el modelo de delegado. Un delegado es un intermediario (o mecanismo similar a un puntero) entre la fuente del evento y el código que maneja el evento. Asocia un delegado con un evento al incluir el tipo de delegado en la declaración del evento
  • Acceso a base de datos: El manejo de base de datos y c# es un requisito importante en la mayoria de los lenguajes. Tambien lo es con c# con sql server
  • Unity: Unity esta basado en codigo c++ y c#. De ese modo para el uso del motor unity se debe tener un alto control en c# y la programacion orientado a objetos.

C++ es la mejora de C. Cabe definir entonces que C++ es la extension de su antecesora permitiendo la manipulacion de objetos. Ademas posee programacion generica( mas centrada en el algoritmo que en los datos). Los algoritmos pueden expresados de la forma más independiente posible de detalles concretos, permitiendo así que puedan servir para la mayor variedad posible de tipos y estructuras de datos. Entre sus caracteristicas estan la sintaxis heredada de C, didactico, compilado, alto nivel, fuertemente tipado y orientado a objetos

  • Didactico: Junto a C es uno de los lenguajes que mas te adapta para aprender otros lenguajes. Es decir que seras capaz de entender las estructuras de codigo de lenguajes como Java, python, javascript, php, etc. Es una de las grandes ventajas de aprender primero este lenguaje
  • Compilado: Al igual que C# y C el codigo se compila, Un compilador suele generar programas más rápidos y eficientes, ya que el análisis del lenguaje fuente se hace una sola vez, durante la generación del programa equivalente. En cambio, un intérprete se ve obligado generalmente a analizar cada instrucción tantas veces como se ejecute (incluso miles o millones de veces).
  • Orientado a objetos: Es una metodología especial de programación de software en la que un programa informático se diseña como la interrelación entre un conjunto de instancias conocidas como objetos. Dichos objetos son entidades que tienen un estado que está descrito por sus variables internas que se conocen como atributos o propiedades; y que tienen comportamientos que están plasmados en funciones o métodos que se utilizan para manipular las variables que describen el estado del objeto. Los objetos que tienen estado y comportamiento idénticos se agrupan o clasifican en "clases". La programación orientada a objetos ofrece los conceptos de encapsulamiento(proceso de almacenar en una misma sección los elementos de una abstracción que constituyen su estructura y su comportamiento; sirve para separar el interfaz contractual de una abstracción y su implantación), abstracción ( expresa las características esenciales de un objeto, las cuales distinguen al objeto de los demás. Además de distinguir entre los objetos provee límites conceptuales), herencia (mecanismo que permite derivar una clase a otra clase. Las clases hijas pueden utilizar tanto sus métodos y propiedades como de la clase padre, siempre que su modificador de acceso lo permita) y polimorfismo ( se refiere a la propiedad por la que es posible enviar mensajes sintácticamente iguales a objetos de tipos distintos).

PHP es un lenguaje de programación destinado a desarrollar aplicaciones para la web y crear páginas web, favoreciendo la conexión entre los servidores y la interfaz de usuario. El código PHP se ejecuta en el servidor que, al leer los comandos, puede activar todos los elementos funcionales y la interfaz visual del sitio web. Entre las caractericas de PHP esta:

  • De facil aprendizaje: El lenguaje PHP se considera uno de los más fáciles de aprender entre las muchas opciones que un programador puede y debe tener en su portfolio de recursos y conocimientos. Se considera el lenguaje con la mejor curva de aprendizaje, siendo superado por Python.
  • Codigo abierto: Los programadores también pueden usar el facilitador de código abierto para mejorar aún más PHP, eliminando cualquier tipo de fallas, bugs o, incluso, trabajando en la optimización del desempeño. Esta es una práctica muy común en la comunidad PHP, que es proactiva a la hora de compartir actualizaciones para que todos puedan usar las versiones mejoradas.
  • Compatibles y amistosos con la gran mayoria de base de datos: Es importante que los sitios web también tengan un buen nivel de compatibilidad con las bases de datos, ya que son una parte fundamental de la estructura. El lenguaje PHP vuelve todo más simple, debido a que resuelve bien el dinamismo para cargar elementos de las páginas con agilidad y sin fallas, sin restringir el rendimiento de la aplicación. PHP soporta varias bases de datos, incluyendo MySQL, Oracle, PostgreSQL, etc., junto con la integración de bases de datos.
  • Multiparadigma: PHP es multiparadigma, cubre la programacion imperativa, funcional, orientado a objetos, procedural, reflexivo.

Este lenguaje de scripting del lado del servidor puede generar contenidos dinámicos de páginas, crear archivos en un servidor, abrir, escribir, leer, cerrar o eliminar archivos.

Además de esto, PHP también puede recoger datos de tus formularios, admitir cookies, realizar la encriptación de datos y controlar el acceso de los usuarios.

Algunos de los sitios web populares que utilizan PHP son WordPress.com, Facebook.com, Wikipedia.org, Zoom.us, Microsoft.com, Canva.com, Salesforce.com, y muchos más.

Habilidades mas buscadas de un programador de PHP:

Pueden acelerar el desarrollo al ofrecer soluciones repetibles a problemas comunes. También ayudan a mejorar la legibilidad del código, ya que otros desarrolladores reconocerán los patrones que se utilizan. Algunos de los patrones de diseño más comunes en aplicaciones PHP incluyen Factory, Strategy y Singleton,
  • conocimientos en Composer, PSR, PHPUnit: Composer es un administrador de dependencias que se encargará de tus bibliotecas, clases y funciones, permitiéndote instalar fácilmente diferentes versiones de cada una por proyecto. Composer también buscará nuevas versiones de estas bibliotecas. PSR es una especificación de PHP diseñada para estandarizar los conceptos de programación de PHP y mejorar la interoperabilidad de los componentes y marcos de PHP. PHPUnit, mientras tanto, es un marco de prueba unitario popular para la prueba automatizada de pequeñas unidades de código con los resultados esperados. Las pruebas unitarias son una buena práctica para cualquier desarrollador de PHP y se pueden aplicar en un enfoque de desarrollo basado en pruebas (TDD), donde las pruebas se configuran incluso antes de que se escriba el código.

Un marco de código abierto desarrollado por Microsoft, .NET proporciona pautas de programación que se pueden utilizar para crear una amplia variedad de aplicaciones. El marco .NET tiene muchas bibliotecas, API y lenguajes diferentes que ayudan a crear aplicaciones. Los diferentes tipos de aplicaciones que se pueden crear con .NET incluyen aplicaciones web, aplicaciones móviles iOS, Windows y Android, juegos 3D, microservicios de Azure, aplicaciones empresariales y mucho más.

.NET es uno de los frameworks más populares y está preparado para seguir siéndolo en el futuro previsible. Estos son algunos de los principales desarrollos que han contribuido a la creciente popularidad de .NET en los últimos años.

  • Alta velocidad: Los sitios web y las aplicaciones creadas con .NET tienden a ser más rápidos. Esto se debe a que las aplicaciones .NET se compilan y no se interpretan. El código compilado significa que el código se traduce en código objeto y luego se ejecuta. Esto tiende a ser más rápido que el código interpretado (usado en PHP) porque el código traducido debe leerse primero antes de ejecutarse.
  • Más control sobre el código: .NET utiliza lenguajes orientados a objetos como VB.NET, C ++ y C #. La principal ventaja de estos lenguajes es que el código tiende a ser más granular. También se presta más fácilmente a pruebas y reutilización.
  • Variedad de aplicaciones: Microsoft siempre ha intentado proporcionar a los desarrolladores las herramientas adecuadas para desarrollar aplicaciones de vanguardia; tanto en términos de tipo de aplicación como de pila tecnológica. Hoy .NET admite una amplia variedad de aplicaciones. Desde aplicaciones web con capacidad de respuesta multiplataforma y juegos 3D hasta aplicaciones empresariales y aplicaciones habilitadas para la nube.

Habilidades mas buscadas de un programador de .NET:

  • Competencia con C #: C # es actualmente uno de los lenguajes más utilizados y admitidos cuando se trata de crear aplicaciones .NET . Es especialmente importante tener un conocimiento sólido de LINQ, funciones asíncronas y conceptos de programación orientada a objetos. Los conceptos importantes de programación orientada a objetos incluyen sobrecarga de herencia, anulación, abstracción y encapsulación. También es imprescindible un conocimiento profundo de Visual Studio de Microsoft, que es el entorno de codificación para C #.
  • Conocimiento del framework .NET: Una de las habilidades más críticas que necesita un buen desarrollador .NET es su familiaridad con el marco .NET. Es importante tener un conocimiento profundo de lo que es capaz de hacer el marco para que sepan dónde buscar cuando están construyendo una solución. El conocimiento de cosas como .NET Framework Runtime y .NET Framework Class Library es crucial. También es importante que un desarrollador de .NET se mantenga actualizado sobre las últimas funciones del marco .NET.
  • Conocimiento de MVC, .NET Core: ASP.NET MVC se ha convertido en uno de los marcos de código abierto más populares en el desarrollo web. C es un lenguaje de programacion primero en su tipo. Aunque se lo cataloga como un lenguaje de alto nivel, se aproxima mas a ser de bajo nivel por su gran complejidad en el manejo. El desarrollo del lenguaje C toma protagonismo para la creación del sistema operativo UNIX que llevarían consigo los nuevos minicomputadores PDP-11, primeros en conectar el procesador, los periféricos y la memoria entre sí mediante un BUS bidireccional. Es imperativo y estruturado y posee algunas caracteristicas importantes:

  • Incluye el concepto de puntero: Un puntero es un objeto del lenguaje de programación, cuyo valor se refiere o apunta a otro valor almacenado en otra parte de la memoria del ordenador utilizando su dirección. Un puntero referencia a una ubicación en memoria, y a la obtención del valor almacenado en esa ubicación se la conoce como desreferenciación del puntero.
  • Es un lenguaje basado en bibliotecas de funciones: Cada programa puede activar otras bibliotecas de funciones, desarrolladas para otros programas, sin tener que desarrollarlas nuevamente.
  • Es un lenguaje compilado: Un programa C se desarrolla en un código fuente. Para ser ejecutado desde la computadora, la fuente debe ser compilada.
  • Facilita la programación modular: Gracias a que cuenta con distintos tipos de almacenamiento de las variables y compilación separada. Además del macro procesador integrado cpp, que permite declarar los encabezados de las funciones y los tipos de datos en archivos con extensión h.
  • Llamadas de funciones por valor: Aunque se pueden pasar punteros a variables para hacer llamados por referencia.

Los desarrolladores de C son responsables de desarrollar el código que se integra en los sistemas operativos base, los módulos del kernel y los sistemas integrados para otros lenguajes de alto nivel. C es un lenguaje de propósito general que se considera un nicho en la programación moderna y, como tal, requiere desarrolladores especializados para escribir el código.
